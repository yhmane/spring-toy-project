### 저장소
저장소는 github -> gitlab (private)으로 이전

##
### 활용 스택

* JDK 1.8
* Gradle
* Spring Boot 2.0
* Rest API (Springfox-swagger-ui)
* Spring-Data-JPA 2.0
* Spring Security, JWT

##
### 예정
* nginx (80 port로 변경), maria db
* 도메인 등록
* https 적용 (443 port)
* ci/cd 도입 (jenkins,  s3, aws code deploy), docker
* slack, trello
* TDD junit5 ( tdd 도입과 함께, swagger -> rest api docs로 변경)
* 프론트 구현 + 배포 (react)
* msa (+ 모듈들 추가)
